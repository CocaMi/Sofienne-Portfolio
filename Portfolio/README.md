## 📦 Getting Started

Clone the repository and run locally:

```bash
git clone https://github.com/CocaMi/Sofienne-Portfolio.git
cd Sofienne-Portfolio/Portfolio
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.
